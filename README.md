es6-classTest
=============

Setup for testing es6 in the browser, but with pre-compilation.

## Usification

```bash
npm i
```

then

```bash
gulp setup
```
then

```bash
gulp
```

Changes in js-files will cause traceur to transpile build.js and then refresh the browser.
